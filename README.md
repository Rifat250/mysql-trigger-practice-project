# MySQL Trigger Practice Project

## Overview

This project contains MySQL trigger practice questions based on a product inventory database.

The project includes:
- stock audit triggers
- price audit triggers
- validation triggers
- business rule triggers
- audit log tables

## Skills Practiced

- MySQL Triggers
- BEFORE INSERT
- BEFORE UPDATE
- AFTER UPDATE
- BEFORE DELETE
- Audit Logging
- SQL Validation
- Database Business Rules
